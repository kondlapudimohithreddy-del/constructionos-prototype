ConstructionOS Clickable Prototype

How to use:
1. Unzip the folder.
2. Open index.html in Chrome or Edge.
3. Click "Open Prototype".
4. Use the left sidebar to navigate.

This is a front-end prototype only. It does not use a real backend or AI yet.
